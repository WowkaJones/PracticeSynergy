# PracticeSynergy
# Проект для производственной практики в Университете "Синергия"

## Описание проекта
Данный репозиторий содержит решение кейс-задачи №3 производственной практики.
Программа реализует класс `WORKER` для учета сотрудников организации и выполняет поиск по стажу работы.

## Функциональность
- Ввод данных о сотрудниках с клавиатуры (ФИО, должность, зарплата, год поступления).
- Хранение данных в динамическом списке (`List<T>`).
- Расчет стажа работы на основе текущего года.
- Поиск и вывод сотрудников, чей стаж превышает заданное пользователем значение.
- Обработка ситуации, если таких сотрудников нет.

## Структура проекта
- `Program.cs` - основной файл программы, содержащий точку входа `Main`.
- `Worker.cs` - модуль с описанием класса `WORKER` (поля, конструкторы, свойства, методы).

## Технологии
- Язык программирования: C#
- Платформа: .NET 8.0
- Среда разработки: Visual Studio 2022
